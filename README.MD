# portfolioku

My personal portfolio — a space where I showcase who I am, what I build, and how I think as a developer.

## 🌐 Live
https://harissfx.github.io/portfolioku

## ⚙️ Built With
- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## 🚀 Run Locally
```bash
git clone https://github.com/harissfx/portfolioku.git
cd portfolioku
npm install
npm run dev
```
