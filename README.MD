# portfolioku

cuman portofolio biasa yang isinya todo app sama landing page doang.  
kalau lu kesini cuma buat liat-liat, gapapa — gue tau ini intimidating. 😇

 **Live:** https://harissfx.is-a.dev/

---

## what's inside

empat hal yang kebanyakan orang pilih salah satu, gue kerjain semua. soalnya dimataku empat hal ini masih satu jalur kalau kamu pelajari btw.

- **3D Modeling** — Blender, FreeCAD & Fusion 360. bukan sekadar muter kubus doang
- **PCB Design** — dari schematic sampe layout board. EasyEDA & KiCad. kalau lu kira ini gampang, coba dulu
- **Website & Dashboard IoT** — web yang beneran ngontrol hardware di dunia nyata. bukan form login doang
- **Custom APK** — Android app custom. karena bilang "bisa mobile dev" terus maksudnya React Native itu lucu

---

## stack

```
React 19 + TypeScript  →  karena nulis vanilla JS di 2025 itu pilihan hidup yang salah
Tailwind CSS           →  Neo-Brutalist. Bootstrap? belum denger
Framer Motion          →  animasi yang bikin orang nanya "pakai apa sih" — pertanyaan favorit gue
ESP32 / Arduino        →  hardware yang beneran nyala. bukan di simulator. nyala beneran
GitHub Pages           →  gratis. karena gue engineer, bukan donatur Vercel
```

---

## run locally

```bash
git clone https://github.com/harissfx/portfolioku.git
cd portfoliando
npm install
npm run dev
```
